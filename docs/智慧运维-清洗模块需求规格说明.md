# 智慧运维 — 清洗计划 / 清洗任务 / 清洗记录 需求规格说明

**文档版本**：1.0  
**编制日期**：2026-03  
**适用模块**：电站运维 → 智慧运维 → 清洗计划、清洗任务、清洗记录

---

## 1. 概述

### 1.1 模块定位

本说明描述智慧运维下三个与光伏组件清洗相关的功能模块的需求规格，三者形成「计划 → 执行（任务填写）→ 归档（记录）」的完整闭环：

| 模块     | 用途简述 |
|----------|----------|
| **清洗计划** | 按电站制定周期性清洗计划，配置执行周期、责任人与任务项模板。 |
| **清洗任务** | 根据计划生成或展示待执行的清洗任务列表，支持清洗人填写每次清洗的现场数据。 |
| **清洗记录** | 对已完成的清洗进行归档查询与详情查看，支持导出。 |

### 1.2 用户角色与入口

- **入口**：顶部选择「电站运维」→ 左侧展开「智慧运维」→ 点击「清洗计划」「清洗任务」或「清洗记录」。
- **主要用户**：运维管理员（制定计划、分配任务）、现场清洗人员（填写任务）、管理人员（查看记录与导出）。

### 1.3 术语

| 术语       | 说明 |
|------------|------|
| 清洗计划   | 针对某一电站的周期性清洗安排，包含周期、起止时间、责任人及任务项模板。 |
| 清洗任务   | 由计划派生的单次执行任务，有任务时间范围与填写状态（未填写/已填写）。 |
| 清洗记录   | 已完成并提交的清洗任务所形成的可查询、可导出的历史记录。 |
| 任务项     | 计划或任务中的检查/填写项，如脏污程度、清洗效果、组件破损等。 |
| 本次第几次清洗 | 同一任务周期内第几次执行清洗的序号（如 1/3 表示第 1 次，共 3 次）。 |

## 2. 清洗计划

### 2.1 功能概述

清洗计划用于为电站创建并管理「按周期重复执行」的清洗计划，包括：计划列表的查询与维护、新增计划（含任务项编辑与周期设置）、以及单条计划的详情/复制/修改/删除。

### 2.2 清洗计划列表页

#### 2.2.1 页面布局

- **顶部筛选与操作区**（同一行，自适应换行）  
  - 筛选：**电站选择**（文本输入，占位「电站选择」）、**查询**按钮。  
  - 操作：**批量删除**、**计划导出**、**新增清洗计划**（主按钮）、**刷新**（图标）。  
- **数据表格**：支持多选（表头复选框全选）、排序列、分页。  
- **分页**：展示「共 N 条记录」、每页条数（如 20 条/页）、页码按钮。

#### 2.2.2 列表字段

| 列名           | 说明                     | 备注     |
|----------------|--------------------------|----------|
| 复选框         | 行多选                   | 表头可全选 |
| 序号           | 当前页序号               | 从 1 递增 |
| 电站名称       | 计划所属电站             | —        |
| 责任人         | 计划责任人姓名           | —        |
| 联系电话       | 责任人联系电话           | —        |
| 清洗周期       | 周期描述（如 90 天）     | —        |
| 计划结束时间   | 计划结束日期/时间        | 可空，显示「-」 |
| 操作           | 详情、复制、修改、删除   | 见 2.2.3 |

#### 2.2.3 列表操作

- **详情**：进入该计划的只读详情（具体页面与字段可单独定义）。  
- **复制**：基于当前计划复制一条新计划，进入编辑态。  
- **修改**：进入该计划的编辑页（与新增共版，数据回填）。  
- **删除**：删除当前计划；可选二次确认。  
- **批量删除**：删除当前勾选的所有计划；可选二次确认。  
- **计划导出**：按当前筛选条件导出计划列表（如 Excel）。  
- **刷新**：重新拉取列表数据。

#### 2.2.4 业务规则

- 查询：根据「电站选择」等条件过滤列表，缺省可展示全部或默认电站。  
- 分页：每页条数可配置（如 20），页码切换时保留筛选条件。

### 2.3 新增 / 编辑清洗计划页

#### 2.3.1 页面入口与返回

- **入口**：列表页点击「新增清洗计划」或某条「修改」/「复制」。  
- **返回**：页面右上角「返回上一页」回到计划列表。

#### 2.3.2 整体布局

- **左侧（约 1/3 宽）**：任务编辑区（任务项列表与增删）。  
- **右侧（约 2/3 宽）**：设置循环周期与起止时间、其他信息（电站、责任人等）。  
- **底部**：**取消**、**保存**按钮。

#### 2.3.3 左侧：任务编辑

- **标题**：「任务编辑」；右侧提供「新增」按钮（可增加自定义任务项或任务组）。  
- **任务分组与任务项**（仅展示名称与操作，不在此处展示选项值）：  
  - **1、脏污与效果**  
    - 子项：脏污程度、主要污染物、清洗效果。  
  - **2、组件破损记录**  
    - 子项：组件是否破损。  
  - **3、其他异常记录**  
    - 子项：是否有其他异常。  
- 每个任务项右侧：**确认（勾选）**、**移除** 操作；用于标识该计划包含哪些项或从模板中移除。  
- 底部可展示「没有更多了」等占位，表示任务项列表结束。

**说明**：此处只定义「该计划包含哪些任务项」，具体选项（如脏污程度：轻微/中等/严重）在「清洗任务」的填写页面中使用。

#### 2.3.4 右侧：设置循环周期、起止时间

| 字段/区域           | 类型       | 说明与规则 |
|--------------------|------------|------------|
| 循环周期           | 数字输入   | 表示「每 N 个周期执行一次」，单位与「循环1次」文案一致（如天/次）。 |
| 第一次任务执行时间 | 日期时间   | 计划首次执行的时间。 |
| 计划结束时间       | 单选+条件  | 选项：「永不」 / 「指定时间」；选「指定时间」时显示日期时间选择器。 |
| 提醒时间           | 文本/选择  | 如「周期结束前」等，用于提醒责任人。 |
| 清洗总次数         | 数字       | 本计划周期内计划执行的清洗总次数（如 3 表示该周期内共 3 次），支持加减按钮与直接输入，最小值 1。 |

#### 2.3.5 右侧：其他

| 字段     | 类型     | 说明 |
|----------|----------|------|
| 电站选择 | 文本/下拉 | 关联电站，必填。 |
| 责任人   | 下拉     | 从系统人员中选择，必填。 |
| 联系电话 | 文本     | 责任人联系电话。 |
| 额外提醒人 | 下拉    | 可选，接收提醒的其他人。 |

#### 2.3.6 保存与取消

- **保存**：校验必填项（如电站、责任人、第一次执行时间等），通过则新增或更新计划并返回列表；失败则提示错误。  
- **取消**：不保存，直接返回列表。

## 3. 清洗任务

### 3.1 功能概述

清洗任务用于展示由计划生成的待执行/已执行任务列表，并支持清洗人在现场「填写清洗任务」：录入脏污与效果、组件破损、其他异常、全流程图片、清洗详情及签字等信息；支持暂存与提交。

### 3.2 清洗任务列表页

#### 3.2.1 页面布局

- **筛选与操作区**  
  - 筛选：**电站选择**、**清洗任务状态**（下拉：请选择 / 未填写 / 已填写）、**清洗任务时间**（开始日期～结束日期）、**查询**。  
  - 操作：**批量删除**、**任务导出**、**刷新**。  
- **数据表格**：多选、分页。  
- **分页**：同清洗计划，如「共 N 条记录」、20 条/页、页码。

#### 3.2.2 列表字段

| 列名             | 说明 |
|------------------|------|
| 复选框           | 行多选 |
| 序号             | 当前页序号 |
| 电站名称         | 任务所属电站 |
| 责任人           | 任务责任人 |
| 联系电话         | 责任人电话 |
| 清洗任务状态     | 未填写 / 已填写（可用不同颜色标签区分） |
| 任务开始时间     | 该任务时间范围开始 |
| 任务结束时间     | 该任务时间范围结束 |
| 任务详情         | 操作：**清洗人填写**、**查看** |

#### 3.2.3 列表操作

- **清洗人填写**：进入「填写清洗任务」页面，并关联当前任务 ID；用于现场录入当次清洗数据。  
- **查看**：进入该任务的只读详情（与「清洗记录详情」结构一致或跳转至记录详情）。  
- **批量删除**：删除勾选任务。  
- **任务导出**：按筛选条件导出任务列表。

### 3.3 填写清洗任务页（表单）

#### 3.3.1 入口与返回

- **入口**：任务列表某行点击「清洗人填写」。  
- **返回**：右上角「返回上一页」回到任务列表。  
- **页头**：标题「填写清洗任务」、修改时间（如 2026/03/17 23:37:02）。

#### 3.3.2 本次第几次清洗

- 在页头右侧区域：**本次为第 [数字输入] 次清洗**。  
- 数字输入：最小值 1，表示本周期内第几次执行清洗（如 1 表示第 1 次，与计划中「清洗总次数」对应，如 1/3 中的「1」）。

#### 3.3.3 任务项填写（与计划任务项对应）

表单分为三个任务组，与计划中「任务编辑」的三组一致；选项与约束如下。

**1、脏污与效果**

| 字段         | 类型   | 选项/规则 |
|--------------|--------|------------|
| 脏污程度     | 下拉   | 请选择、轻微、中等、严重。 |
| 主要污染物   | 下拉   | 请选择、积灰、鸟粪、泥沙、油污、其他；选「其他」时显示补充文本框。 |
| 清洗效果     | 单选   | 达标、未达标；选「未达标」时必填「未达标原因」文本。 |

**2、组件破损记录**

| 字段         | 类型   | 选项/规则 |
|--------------|--------|------------|
| 组件是否破损 | 单选   | 无、有破损。 |
| （当「有破损」时展示） | | |
| 破损位置     | 组合输入 | 第 [串] 串第 [块] 块组件 / [区域组件] 文本。 |
| 破损类型     | 下拉   | 请选择、玻璃裂纹、边框变形、背板破损、其他；选「其他」可填补充。 |
| 破损图片标注 | 图片上传 | 最多 10 张。 |
| 处理建议     | 下拉   | 请选择、无需处理、待更换、临时修复、其他；选「其他」可填补充。 |

**3、其他异常记录**

| 字段         | 类型   | 选项/规则 |
|--------------|--------|------------|
| 是否有其他异常 | 单选 | 无、有异常。 |
| （当「有异常」时展示） | | |
| 异常描述     | 多行文本 | 最多 200 字，显示字数统计。 |
| 异常位置     | 组合输入 | 第 [串] 串第 [块] 块组件 / [区域组件]。 |
| 异常图片标注 | 图片上传 | 最多 10 张。 |
| 处理结果     | 多行文本 | 最多 200 字，显示字数统计。 |

#### 3.3.4 清洗全流程图片

- 三个固定区块，每个区块独立上传：  
  - **清洗前图片（组件原始状态）**：最多 10 张。  
  - **清洗中图片（作业过程）**：最多 10 张。  
  - **清洗后图片（清洁完状态）**：最多 10 张。  
- 每块提供上传入口与数量提示（如「点击上传，最多10张」）。

#### 3.3.5 清洗详情

| 字段     | 类型   | 选项/规则 |
|----------|--------|------------|
| 清洗方式 | 下拉   | 请选择、人工、高压水枪、机器人。 |
| 作业时间 | 日期   | 年月日选择。 |
| 天气情况 | 下拉   | 请选择、晴、阴、多云。 |

#### 3.3.6 确认签字

| 项目               | 说明 |
|--------------------|------|
| 清洗负责人签字及日期 | 签字区域（手写或上传签名）+ 日期选择；提供「清空」「上传签名」按钮。 |
| 验收人签字及日期   | 同上，独立签字与日期。 |

#### 3.3.7 底部操作

- **暂存**：保存当前填写内容，不提交，任务状态仍为「未填写」，可再次进入继续填。  
- **提交清洗结果**：校验必填项后提交，任务状态变为「已填写」，并关闭表单返回列表。

#### 3.3.8 业务规则

- 提交时可根据需要校验：本次第几次清洗、必填任务项、至少一张全流程图片等（具体以产品规则为准）。  
- 暂存数据与任务 ID 绑定，再次进入「清洗人填写」时回填已暂存内容。

## 4. 清洗记录

### 4.1 功能概述

清洗记录用于按条件查询已完成或已提交的清洗记录，并支持查看单条记录的只读详情及导出（如 PDF）。

### 4.2 清洗记录列表页

#### 4.2.1 页面布局

- **筛选与操作区**  
  - 筛选：**电站选择**、**清洗时间**（开始日期～结束日期）、**查询**。  
  - 操作：**批量删除**、**记录导出**、**刷新**。  
- **数据表格**：多选、分页。

#### 4.2.2 列表字段

| 列名     | 说明 |
|----------|------|
| 复选框   | 行多选 |
| 序号     | 当前页序号 |
| 电站名称 | 记录所属电站 |
| 清洗时间 | 该次清洗的日期（可空显示「--」） |
| 责任人   | 责任人姓名 |
| 清洗人员 | 参与清洗的人员，多人在列表中以标签形式展示（如 顾佳兴、周焱） |
| 操作     | **详情**（打开只读详情页） |

#### 4.2.3 列表操作

- **详情**：打开「清洗记录详情」页，展示该条记录的全部填写内容（只读）。  
- **批量删除**：删除勾选记录（逻辑与权限由后端/产品定义）。  
- **记录导出**：按筛选条件导出记录列表。

### 4.3 清洗记录详情页（只读）

#### 4.3.1 入口与返回

- **入口**：记录列表某行点击「详情」。  
- **返回**：右上角「返回上一页」回到记录列表。  
- **页头**：标题「清洗记录详情」；右侧可展示「本次为第 M/N 次清洗」、**导出 PDF**、**返回上一页**。

#### 4.3.2 内容结构（与「填写清洗任务」一致，全部只读）

- **清洗项**（原「任务项填写」）  
  - 1、脏污与效果：脏污程度、主要污染物、清洗效果（及未达标原因）以只读文本/标签展示。  
  - 2、组件破损记录：组件是否破损；若有破损，破损位置、类型、图片、处理建议只读展示。  
  - 3、其他异常记录：是否有其他异常；若有，异常描述、位置、图片、处理结果只读展示。  
- **清洗全流程图片**  
  - 清洗前/清洗中/清洗后三块，仅展示已上传图片，无上传入口。  
- **清洗详情**  
  - 清洗方式、作业时间、天气情况，只读文本展示。  
- **确认签字**  
  - 清洗负责人签字及日期、验收人签字及日期，只读展示签字与日期。

#### 4.3.3 导出 PDF

- 点击「导出 PDF」可生成当前详情页的 PDF 文件，供存档或外发。

## 5. 数据与状态约定（参考）

### 5.1 清洗计划

- 计划唯一标识：id。  
- 关键属性：电站名称、责任人、联系电话、清洗周期、计划结束时间；与「任务编辑」中任务项配置、循环周期、第一次执行时间、清洗总次数等一致。

### 5.2 清洗任务

- 任务唯一标识：id。  
- 关键属性：电站名称、责任人、联系电话、清洗任务状态（未填写/已填写）、任务开始时间、任务结束时间。  
- 与计划的关联：通过计划 id 或电站+周期等业务键关联；任务列表可由计划自动生成或人工创建。

### 5.3 清洗记录

- 记录唯一标识：id。  
- 关键属性：电站名称、清洗时间、责任人、清洗人员（多人）。  
- 与任务的关系：通常由「已填写」的清洗任务提交后生成一条记录，或任务与记录为一对一关系以状态区分。

---

## 6. 非功能需求（建议）

- **性能**：列表分页加载，单页条数建议 20～50；详情页图片懒加载。  
- **兼容**：与整体平台一致，支持现代浏览器与常见分辨率。  
- **权限**：列表与导出可按角色控制（如仅管理员可删除、导出）。  
- **审计**：关键操作（如计划删除、任务提交）可记操作日志。
